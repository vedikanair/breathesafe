BreatheSafe
Real-time air quality monitoring dashboard for India. Track AQI across cities, explore pollutant breakdowns, and view station-level data on an interactive map.

Tech Stack
Frontend- Next.js (App Router), TypeScript, Tailwind CSS, Recharts
Backend- Supabase (PostgreSQL + Views)
Data- OpenAQ API v3

Features:
City-level AQI cards with avg, peak, and lowest readings
State-level AQI heatmap across India
Pollutant comparison chart (PM2.5, PM10, NO2, SO2, CO, O3)
Top danger zones ranked by AQI
Per-station detail pages with pollutant history
